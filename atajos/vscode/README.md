# VScode

Sirve para escribir, depurar y compilar código en diversos lenguajes de programación. Algunas de sus principales ventajas son:

- **Multiplataforma**: Funciona en Windows, macOS y Linux, por lo que los desarrolladores pueden usarlo independientemente de su sistema operativo.

- **Ligero y personalizable**: A pesar de ser un IDE (Entorno de Desarrollo Integrado) completo, es muy liviano y puede personalizarse a través de extensiones para adaptarlo a las necesidades específicas.

- **Amplia gama de lenguajes**: Soporta de forma nativa lenguajes como JavaScript, TypeScript, Node.js, Python, C++, C#, Java, PHP, Go y muchos más.

- **Depuración integrada**: Permite depurar código directamente dentro del editor, lo que facilita encontrar y corregir errores.

- **Control de versiones integrado**: Trae integración nativa con sistemas de control de versiones como Git y GitHub.

- **IntelliSense**: Ofrece autocompletado de código inteligente, resaltado de sintaxis y refactorización de código para aumentar la productividad.

- **Terminal integrado**: Incluye una terminal de comandos incorporada para ejecutar tareas y comandos relacionados con el proyecto.

- **Extensiones poderosas**: Existe un vasto ecosistema de extensiones que amplían las capacidades de VSCode, como resaltado de sintaxis, linters, snippets y más."

- **Comillas o paréntesis al seleccionar texto**: rodea con par de caracteres

- **Ctrl/Cmd + Shift + K**: Elimina la línea actual.

- **(Shift + Alt + Flecha arriba/abajo / Ctrl + Shift + D)**: Duplica la línea actual hacia arriba o abajo.

- **Ctrl/Cmd + Enter**: Inserta una línea en blanco debajo del cursor.

- **Ctrl/Cmd + Shift + Enter**: Inserta una línea en blanco arriba del cursor.

- **Ctrl/Cmd + Shift + { }**: Ir al paréntesis, corchete o llave correspondiente.

- **Alt + Shift + Flecha Izq/Der**: Expande o contrae la selección (paréntesis, corchete o llave, comillas)

- **Ctrl/Cmd + D**: Selecciona la siguiente aparición de la palabra actualmente seleccionada.

- **Ctrl/Cmd + Shift + L**: Selecciona todas las apariciones de la palabra actualmente seleccionada.

- **Ctrl/Cmd + Sfift + 7**: Comentar / Descomentar  las líneas seleccionadas.

- **Alt + Clic**: Inserta múltiples cursores en la misma posición."
